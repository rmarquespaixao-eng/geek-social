## Troubleshooting

**`port is already allocated`** â€” outro processo ocupa 5432/8080/9000. Mate ou mude a porta no compose.

**`role "dev" does not exist`** â€” primeiro boot falhou no setup. `docker compose down -v` + `up` recria.

**Geek Social retorna 403 em URLs do MinIO** â€” `STORAGE_PUBLIC_URL` precisa incluir `/geek-social-media` no path. Sem isso, URLs ficam `http://localhost:9000/posts/abc.jpg` (errado) em vez de `http://localhost:9000/geek-social-media/posts/abc.jpg` (correto).

**Migration falha com "database does not exist"** â€” esqueceu o passo 1 (criar `geek_social` Ă  mĂŁo).
